# Netflix Movie Data Analysis

## Project Overview
This project analyzes a Netflix Movie dataset containing more than 9000 records using Python. The objective is to explore movie trends, genres, ratings, popularity, and release patterns through data analysis and visualization.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project objectives 
• Clean and preprocess the dataset.
• Perform Exploratory Data Analysis(EDA)
• Analyze genre distribution.
• Find the highest and lowest-rated Movies.
• Analyze popularity trends.
• Identify yearly movie release patterns.

## Key Insights 
- Most frequent movie genre.
- Highest-rated movie.
- Most popular movie.

## Skill Demostrated 
- Data cleaning
- Data Visualization
- Data Interpretation
- EXploratory Data Analysis(EDA)
